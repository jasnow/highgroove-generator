# Highgroove-Generator

Generate Rails projects the way we like them.

## Installation

```bash
 gem install highgroove_generator
```

## Usage

```bash
 # create a new project
 highgroove new NAME

 # specify the database type
 highgroove new --database=DATABASE

# get help
highgroove help
```
